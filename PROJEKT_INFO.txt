EMILS DASHBOARD – PROJEKTINFO
==============================
Repo:      https://github.com/emillofkvist/emils-dashboard

FILER
-----
- index.html    : HTML + CSS
- config.js     : API-URL:er, CORS-proxies, RSS-feeds
- dashboard.js  : All hämtning och rendering

GIT-WORKFLOW (varje gång du gör ändringar)
-------------------------------------------
  OBS: GitHub Pages serverar direkt från master-grenen – ingen gh-pages-branch behövs.

  1. git add config.js dashboard.js index.html PROJEKT_INFO.txt
  2. git commit -m "Beskrivning av ändringen"
  3. git push origin master

  Ändringen syns live på GitHub Pages inom ~1 minut efter steg 3.

API-ENDPOINTS (vad koden faktiskt använder)
--------------------------------------------
1. Väder (SMHI snow1g/v1):
   https://opendata-download-metfcst.smhi.se/api/category/snow1g/version/1/...
   Förväntar: timeSeries[].data.{air_temperature, wind_speed, relative_humidity, symbol_code}
   OBS: pmp3g/v2 lades ned 31 mars 2026

2. Elpris SE4:
   https://www.elprisetjustnu.se/api/v1/prices/{year}/{mm}-{dd}_SE4.json
   Förväntar: array med SEK_per_kWh och time_start

3. Börskurser (Yahoo Finance via api.allorigins.win):
   https://api.allorigins.win/raw?url=https%3A%2F%2Fquery1.finance.yahoo.com%2Fv8%2Ffinance%2Fchart%2F{symbol}%3F...
   Symboler: ^OMX, ^GSPC, ^IXIC, ^DJI
   Förväntar: chart.result[0].indicators.quote[0].close[]
   OBS: Yahoo Finance begränsar server-side requests – kan ge "Ej tillgänglig" vid blockering

4. Google Kalender (iCal via api.allorigins.win):
   https://api.allorigins.win/raw?url={encoded-ical-url}
   Förväntar: iCal-text med BEGIN:VEVENT-block

5. Nyheter SVT (RSS via api.allorigins.win):
   https://api.allorigins.win/raw?url=https%3A%2F%2Fwww.svt.se%2Fnyheter%2Frss.xml
   Förväntar: RSS XML med <item>-element

6. Nyheter DN (RSS via api.allorigins.win):
   https://api.allorigins.win/raw?url=https%3A%2F%2Fwww.dn.se%2Frss%2F
   Förväntar: RSS XML med <item>-element

7. AI-nyheter TechCrunch AI (RSS via api.allorigins.win):
   https://api.allorigins.win/raw?url=https%3A%2F%2Ftechcrunch.com%2Fcategory%2Fartificial-intelligence%2Ffeed%2F
   OBS: ersatte openai.com/news/rss.xml som inte längre existerar

8. AI-nyheter The Verge AI (Atom-feed via api.allorigins.win):
   https://api.allorigins.win/raw?url=https%3A%2F%2Fwww.theverge.com%2Frss%2Fai-artificial-intelligence%2Findex.xml
   OBS: Atom 1.0-format (<entry>/<link href="">/<published>) – koden hanterar båda RSS och Atom

9. Macworld Sverige (RSS via api.allorigins.win):
   https://api.allorigins.win/raw?url=https%3A%2F%2Fwww.macworld.se%2Ffeed

CORS-PROXYER
------------
  Allt RSS/börs/kalender: https://api.allorigins.win/raw?url={encoded_full_url}
  OBS: corsproxy.io kräver betalplan (returnerar 200 med error-JSON) – använd INTE
  OBS: cors.eu.org kräver nu allowlist (returnerar 403 host_not_allowed för allt) – använd INTE
