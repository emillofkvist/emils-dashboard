GIT-WORKFLOW (varje gång du gör ändringar)
-------------------------------------------
  1. git add config.js dashboard.js index.html PROJEKT_INFO.txt
  2. git commit -m "Beskrivning av ändringen"
  3. git push origin master
  4. git checkout gh-pages
  5. git merge master
  6. git push origin gh-pages
  7. git checkout master

Ändringen syns live på GitHub Pages inom ~1 minut efter steg 6.

API-ENDPOINTS (vad koden faktiskt använder)
--------------------------------------------
1. Väder (SMHI snow1g/v1):
   https://opendata-download-metfcst.smhi.se/api/category/snow1g/version/1/...
   Förväntar: timeSeries[].data.{air_temperature, wind_speed, relative_humidity, symbol_code}
   OBS: pmp3g/v2 lades ned 31 mars 2026

2. Elpris SE4:
   https://www.elprisetjustnu.se/api/v1/prices/{year}/{mm}-{dd}_SE4.json
   Förväntar: array med SEK_per_kWh och time_start

3. Börskurser (Yahoo Finance via corsproxy.io):
   https://corsproxy.io/?<encoded-yahoo-url>
   Symboler: ^OMX, ^GSPC, ^IXIC, ^DJI
   Förväntar: chart.result[0].indicators.quote[0].close[]

4. Google Kalender (iCal via corsproxy.io):
   https://corsproxy.io/?<encoded-ical-url>
   Förväntar: iCal-text med BEGIN:VEVENT-block

5. Nyheter SVT (RSS via corsproxy.io):
   Förväntar: RSS XML med <item>-element

6. Nyheter DN (RSS via corsproxy.io):
   Förväntar: RSS XML med <item>-element

7. AI-nyheter TechCrunch AI (RSS via corsproxy.io):
   https://techcrunch.com/category/artificial-intelligence/feed/
   OBS: ersatte openai.com/news/rss.xml som inte längre existerar

8. AI-nyheter The Verge AI (RSS via corsproxy.io):
   https://www.theverge.com/rss/ai-artificial-intelligence/index.xml

9. Macworld Sverige (RSS via corsproxy.io):
   https://www.macworld.se/feed

CORS-PROXYER
------------
  Allt RSS/börs/kalender: https://corsproxy.io/?{encoded_url}
  OBS: allorigins.win är nere (522-fel) – använd INTE den
