API-ENDPOINTS (vad koden faktiskt använder)
--------------------------------------------
1. Väder (SMHI pmp3g/v2):
   https://opendata-download-metfcst.smhi.se/api/category/pmp3g/version/2/geotype/point/lon/12.8167/lat/56.1167/data.json
   Förväntar: timeSeries[].parameters[] med t, ws, r, Wsymb2

2. Elpris SE4:
   https://www.elprisetjustnu.se/api/v1/prices/{year}/{mm}-{dd}_SE4.json
   Förväntar: array med SEK_per_kWh och time_start

3. Börskurser (Yahoo Finance via api.allorigins.win):
   https://api.allorigins.win/raw?url=https://query1.finance.yahoo.com/v8/finance/chart/{symbol}?interval=1d&range=2d
   Symboler: ^OMX, ^GSPC, ^IXIC, ^DJI
   Förväntar: chart.result[0].indicators.quote[0].close[]

4. Google Kalender (iCal via corsproxy.io):
   https://corsproxy.io/?<encoded-ical-url>
   Förväntar: iCal-text med BEGIN:VEVENT-block

5. Nyheter SVT (RSS via api.allorigins.win):
   https://api.allorigins.win/raw?url=https://www.svt.se/nyheter/rss.xml
   Förväntar: RSS XML med <item>-element

6. Nyheter DN (RSS via api.allorigins.win):
   https://api.allorigins.win/raw?url=https://www.dn.se/rss/
   Förväntar: RSS XML med <item>-element

7. AI-nyheter OpenAI (RSS via api.allorigins.win):
   https://api.allorigins.win/raw?url=https://openai.com/news/rss.xml
   OBS: Oklar URL – OpenAI har bytt URL-struktur flera gånger

8. AI-nyheter The Verge AI (RSS via api.allorigins.win):
   https://api.allorigins.win/raw?url=https://www.theverge.com/rss/ai-artificial-intelligence/index.xml
   Förväntar: RSS XML med <item>-element

9. Macworld Sverige (RSS via api.allorigins.win):
   https://api.allorigins.win/raw?url=https://www.macworld.se/feed
   Förväntar: RSS XML med <item>-element
